Compile:
  Place
    goatmalloc.c,
    goatmalloc.h,
    Makefile, and
    test_goatmalloc.c
  into the same directory.

  Run
    "make":
    same as "make all".

    "make all":
    same as "make test"

    "make test":
    compiles project and creates "test_goatmalloc"

    "make clean":
    deletes "test_goatmalloc"

Usage:
  Do
    "./test_goatmalloc"
  to run the all test cases for the project.
